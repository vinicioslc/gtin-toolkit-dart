# gtin-toolkit-dart

A toolkit for validate GTIN's and generate GTIN as well.

## How to

- Import

```dart

import 'package:gtin_toolkit/gtin_toolkit.dart' as gtinToolkit;

```

- Use

```dart

gtinToolkit.validate('838393949495') // => true

gtinToolkit.validate('8383939494952') // => false

gtinToolkit.validate('8383939494952') // => false

```
